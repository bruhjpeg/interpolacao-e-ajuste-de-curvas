# interpolacao-e-ajuste-de-curvas

Este repositório contém implementações em Python para três métodos numéricos:

- Interpolação de Lagrange
- Interpolação de Newton
- Ajuste de curvas por mínimos quadrados

## Como usar

Execute o exemplo abaixo:

```bash
python exemplo.py
```

As funções estão disponíveis em [metodos.py](metodos.py).
